# 1.セキュアなファイル共有プラットフォーム
ユーザーがファイルをアップロードし、特定の人と安全に共有できるプラットフォーム。

ファイルは暗号化され、アクセス制御が厳密に管理される。

## 技術

 - [x] JWT
 - [ ] OAuth 2.0

セキュリティ:
 - [x] AES暗号化
 - [ ] HTTPS
 - [x] CSRF対策

パフォーマンス:
 - [x] CDN（Content Delivery Network）
 - [x] キャッシュ戦略

フロントエンド:
 - [x] React

 - [x] Go
